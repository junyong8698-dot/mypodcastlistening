I've been building a systematic BTC trading system using multi-agent AI (Claude via OpenClaw + Telegram). Instead of one monolithic strategy, I split responsibilities across specialized agents:

- **Regime detection agent** — classifies market as trending/ranging/volatile using macro indicators
- **R&D agent** — runs research experiments autonomously, proposes strategy changes
- **9-member AI board** — votes on proposed changes weekly (yes, like a real board meeting)

The whole thing runs on Telegram. I get daily reports, can query any agent for reasoning, and the board minutes are auto-generated.
